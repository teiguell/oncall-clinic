## Decisions flagged

1. **308 (Next.js permanent: true)** en lugar de 301 — ambos
   transfieren SEO equity, 308 además preserva el método HTTP. Mismo
   outcome para nuestro caso.
2. **AdministrativeArea para Costa del Sol / Costa Blanca** —
   schema.org no tiene un type "coastal region" exacto;
   AdministrativeArea es el más cercano (subordinada a Spain).
3. **`reviewCount: 127`** sigue siendo placeholder (no live data
   todavía en consultation_reviews).
4. **Lead notifications → tei@oncall.clinic** — para alpha. Cuando
   escale, sugerir cambio a leads@ shared inbox.
5. **Specialty enum corto** (general/pediatrics/gynecology/other) —
   ampliar requiere actualizar API allow-list + `<select>` options
   + i18n strings. Los 3 deben moverse juntos.
6. **/blog robots noindex,follow** mientras no haya artículos.
   Cuando shippeen los primeros, flippear a index,follow vía
   per-article generateMetadata.
7. **/medico-domicilio/madrid → /medicos** (no a /) — más relevante
   para la query "médico a domicilio Madrid" dado que no servimos
   esas ciudades.
