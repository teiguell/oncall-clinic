```

HTTP/2 307
x-matched-path: /es/pro/registro

$ curl -sS -I https://oncall.clinic/en/pro/registro

```

| URL | HTTP | Body redirect target |
|---|---|---|
| `/es/pro/registro` | 307 | `/es/doctor/onboarding` |
| `/es/pro/login` | 307 | `/es/login?role=doctor` |
| `/es/pro/dashboard` | 307 | `/es/doctor/dashboard` |
| `/en/pro/registro` | 307 | `/en/doctor/onboarding` |

**Note on the 307-without-Location pattern:** Next.js 14 App Router `redirect()`
in a server component returns a 307 with the redirect target encoded in the
RSC payload (and a `<html id="__next_error__">` fallback HTML body), not in a
classic `Location:` header. Browsers handle this correctly via the Next.js
runtime; curl shows 307 + body. Decoded body confirms `doctor/onboarding;307`
present in `/es/pro/registro` response. ✅

## Decisions flagged for Director

1. **307 vs 308 on alias redirects.** I documented the redirects as 308
   (permanent + method-preserving) in the file headers because that's
   the semantic intent of a "stable alias." However, `redirect()` from
   `next/navigation` defaults to 307 (temporary). Switching to 308 would
   require `redirect(target, 'replace')` or use of
   `permanentRedirect()`. **Question:** OK with 307, or do you want
   permanentRedirect for SEO? My reading: 307 is fine — the doctor
   onboarding URL might change before launch, and search engines treat
   307 fine for the brief alpha window.

2. **eta_sms_sent_at column.** The doctor ETA update endpoint uses
   the 60s rate limit as a soft idempotency guarantee. For strict
   "fire exactly once when crossing the 10-min threshold" semantics
   we'd need a `consultations.eta_sms_sent_at` column. Punted to a
   follow-up so Round 14-C ships without a schema change.

3. **Twilio trial limitation.** Only `+34 681 XX XX XX` (your
   verified number) receives SMS in trial mode. Production go-live
   needs $20 deposit on Twilio. The first patient SMS to a real
   booking will silently fail with error code 21608 until then —
   the log row will show `status='failed', error_code='21608'`.
